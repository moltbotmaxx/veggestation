# Veggestation

Veggestation es una PWA mobile-first para acompanar a una persona desde la siembra hasta la cosecha de una verdura o hierba comestible incluida en un kit fisico.

La idea base es vender un kit sencillo para principiantes y usar la app como acompanamiento:

- que le diga al usuario cuando regar,
- cuanta luz necesita,
- que cambio deberia esperar en cada etapa,
- y que hacer si algo no va bien.

## Punto de partida recomendado

Para un primer kit, la mejor mezcla entre facilidad, progreso visible y bajas probabilidades de frustracion es:

- rabano,
- lechuga de hoja,
- albahaca,
- frijol de mata.

Tomate y chile dulce son atractivos, pero para una primera version suelen requerir mas espacio, mas sol, soporte y una experiencia de cuidado menos indulgente para principiantes.

## Principio de producto

Veggestation no necesita empezar como una app "inteligente". La v1 puede ser muy buena si hace bien tres cosas:

1. Guiar.
2. Recordar.
3. Dar tranquilidad.

El objetivo no es diagnosticar cualquier problema botanico; es lograr que mas usuarios lleguen a germinar, cuidar y cosechar con exito.
